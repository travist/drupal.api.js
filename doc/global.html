get(object, endpoint, query, callback, cache)

API function to get any results from the drupal API. Return the object. drupal.api.get(entity, function(object) { console.log(object); }); Return a list of events within an entity. drupal.api.get(entity, 'events', function(events) { console.log(events); }); Return a list of nodes with type='page'. drupal.api.get({}, {type:'page'}, function(object) { }); Return a list of events provided a query within a node. drupal.api.get(entity, 'events', {month: 6}, functoin(events) { console.log(events); });
